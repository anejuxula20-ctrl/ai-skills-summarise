技术判断

高 star velocity 是早期信号，不是采购结论；更适合用来发现主题迁移、生态入口和需要提前防守的风险。

Agent 变成能力栈

Hermes、ECC 和 skills 类仓库的增长说明开发者正在把 agent 当作可扩展运行时，而不是单一聊天机器人。

内容工具继续吸星

短视频、社媒卡片和封面生成把 AI 输出直接连接到可发布产物，热度来自非常明确的交付价值。

代码理解是底座

大型代码库与知识库的图谱化、可问答化正在成为 agent 真正进入工程工作的基础能力。

技能安全会升温

agent skills 可能影响文件、命令和外部服务访问，安装前审计、白名单和沙箱会越来越重要。
